Recall Rumble

A memory based game where you can submit a set of terms and definitions,
then play a game of memory, matching those terms to their said definitions.
A fun and efficient way to study and increase your knowledge!

About the project: This project was based on the game of memory that we all played and loved as a kid. The rules of the game are simple and easy to understand, and strikes a strand of nostalgia within us all.

How to use:
First, navigate to the "Make A Set" tab in the top left. From there, you can enter terms and definitions to create a set of flash cards. Once you have entered all your cards, navigate to the "Memory Game" tab in the navigation bar at the top of the page. Once there, you can being playing the memory game using your flash cards. Enjoy!
